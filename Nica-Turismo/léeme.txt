En este espacio vamos a trabajar el proyecto del hackathon,
cada una tendrá su propia rama donde subir sus avances, y luego
yo los enviaré a la rama principal.

Comandos útiles de git:

git init
Crea un nuevo repositorio en la carpeta actual.

git clone <url>
Copia un repositorio remoto en tu máquina.

git status
Muestra el estado de los archivos (nuevos, modificados, listos para confirmar).

git add <archivo>
Prepara un archivo para ser confirmado en el próximo commit.

git commit -m "mensaje"
Guarda los cambios en el historial del repositorio con un mensaje.

git log
Muestra el historial de commits.

git push origin main
Sube tus commits al repositorio remoto (ejemplo: rama main).

git pull origin main
Descarga y combina los cambios más recientes del repositorio remoto.

git branch
Lista las ramas existentes en el proyecto.

git checkout -b <nombre-rama>
Crea y cambia a una nueva rama.
