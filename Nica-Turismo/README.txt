Descripción del proyecto
"Nica Turismo" es un sistema web que tiene como propósito facilitar la manera en la que los turistas acceden a los distintos servicios ofrecidos para poder visitar el país. Muchos de estos servicios están divididos en múltiples plataformas, lo que puede hacer tedioso y tardado todo el proceso de búsqueda, obstaculizando la experiencia. Por medio de Nica Turismo, se busca ofrecer a los usuarios todos los servicios necesarios en una sola plataforma, permitiendo así que puedan organizar su viaje con mayor eficiencia. La aplicación ofrecerá servicios confiables, únicos y seguros, mejorando aún más la experiencia del recorrido por territorio nicaragüense.

Requerimientos técnicos
1. Requerimientos mínimos:
- Dispositivo: PC, laptop, tablet o smartphone
- Procesador: Cualquier procesador moderno básico
- RAM: 2 GB
- Almacenamiento: Espacio suficiente para caché temporal
- Sistema operativo: Windows 7+, macOS 10.12+, Android 6+, iOS 11+
- Navegador: Cualquier navegador moderno y actualizado
- Conexión a Internet: 1–5 Mbps (básica)
- Otros: JavaScript habilitado, soporte HTTPS

2. Requerimientos recomendados:
- Procesador: Dual-core o superior
- RAM: 4 GB o más
- Almacenamiento: 10 GB o más disponible
- Pantalla: Resolución mínima 1366×768
- Conexión a Internet: 10 Mbps o más para multimedia
- Otros: Navegador actualizado a la última versión, soporte multimedia (video, audio, animaciones)

Instalación:
- No requiere instalación.
- El acceso se realiza directamente desde cualquier navegador web compatible, ingresando la URL del sitio.
- Solo se necesita conexión a Internet y un dispositivo con los requerimientos mencionados.

Método de uso:
1. Acceso al sitio:
   - Abrir un navegador web compatible.
   - Ingresar la URL del sitio en la barra de direcciones.

2. Navegación por el contenido:
   - Visualizar landing page; se recomienda registrarse o iniciar sesión para poder hacer las reservas.
   - Registrar nombre de usuario, correo electrónico y contraseña. En caso de contar con una cuenta en el sitio, iniciar sesión.
   - Explorar las secciones del sitio: Lugares a visitar, rutas turísticas, transporte, hoteles, perfil de usuario.
   - Utilizar menús, botones y enlaces para acceder a la información deseada.

3. Búsqueda de información específica:
   - Usar el buscador interno para encontrar destinos, tours o servicios concretos.
   - Filtrar resultados por ruta turística, precio y horario.

4. Interacción con servicios del sitio:
   - Completar formularios de reserva de hoteles, transporte y servicios.
   - Descargar materiales disponibles (folletos en PDF).

5. Seguridad y recomendaciones:
   - Asegurarse de tener conexión segura (HTTPS) al ingresar datos personales.
   - Mantener actualizado el navegador para una correcta visualización y funcionamiento del sitio.
